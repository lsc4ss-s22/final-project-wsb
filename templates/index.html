My Wall Street Bets API

This API provides you with instructions on how to access the table containing posts from r/wallstreetbets.
It allows you to index the database based on a date range, author or category.

Example Date Query: URL_BASE/api/start_date:2022-01-01&end_date:2022-02-02

  • Dates are provided in %Y-%m-%d format

    • Example Category Query: URL_BASE/api/category:DD

    Available categories are

  • "Daily Discussion": Daily Discussion Threads
  • "Meme": Meme Post
  • "Gain": User posting stock market gains
  • "Loss": User posting stock market losses
  • "Discussion": Users discussing a topic
  • "Chart": User posting a Chart
  • "DD": User posting Due Diligence on a particular company/market etc.
  • "News": News Articles

    • Example Author Querty: URL_BASE/api/author:FluxRevived
